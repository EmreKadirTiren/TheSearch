# TheSearch
The Search is a HTML game we made for the campfire game jam Eindhoven!
It is a story about mermaid Qiqi who lost her fellow mermaid Heleb.

Go through parkour and boss fights to save Heleb from Ferran the frightening trumpetfish.

Controls:

Moving: use the arrows!

Interacting with things is E

Attacking is spacebar

Controller is not supported but you can play with it.

Have fun and enjoy :)
