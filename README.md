# TheSearch

A lightweight client-side search web application built with plain HTML, CSS, and JavaScript.

## Features

- Real-time keyword search across a local dataset
- Clean, responsive search UI
- No external dependencies

## Getting Started

Open `index.html` in your browser to run the app locally.

## Project Structure

```
index.html   – Main page and search interface
style.css    – Stylesheet
app.js       – Search logic and DOM interaction
```
